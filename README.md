# ranobehub-parser

Parser for get data from [ranobehub](https://ranobehub.com)

| METHOD  | ENDPOINT      | PARAMETERS                                         | DESCRIPTION                |
| ------- | ------------- | -------------------------------------------------- | -------------------------- |
| GET     | `/novela/:id` | `id` (path parameter)                              | Get novel by id            |
| GET     | `/search`     | `page` (query parameter), `take` (query parameter) | Search for novels          |
| GET     | `/chapter`    | `url` (query parameter)                            | Get chapter details by url |
